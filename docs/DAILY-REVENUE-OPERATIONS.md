# Daily Revenue Operations — DocMind

## North Star
- Weekly: >= $1000
- Daily tactical target: 3 to 5 paid customers/day (until stable)

## Daily Execution Blocks (no ads)

### Block A — Acquisition (09:00-11:00)
- Publish 1 TikTok demo (PT-MZ)
- Publish 1 short post on Facebook groups
- Send 15 direct outreach messages (WhatsApp/Facebook)

### Block B — Activation (13:00-15:00)
- Help new users finish first analysis within 10 minutes
- Collect objections and update FAQ/sales copy
- Follow up all users who clicked but did not pay

### Block C — Conversion (18:00-20:00)
- Send limited-time offer message (today only)
- Trigger social proof (screenshots/testimonials)
- Ask all active leads for direct close

## Offer Structure
- Core hook: "Upload 1 doc, get result in 2 minutes"
- Value pack: summary + simplification + suggestions + improved version
- Payment options: M-Pesa, eMola, Visa/Mastercard

## KPI Checklist (end of day)
- Messages sent
- Conversations started
- Signup count
- First analysis count
- Checkout starts
- Paid conversions
- Revenue (MZN + USD)

## Tight Feedback Loop
Every day:
1. Keep top 2 channels, cut worst channel
2. Keep top 2 messages, rewrite weakest message
3. Keep top 1 offer, simplify all others
